Q

Which Marley made the most albums?

Generated SQL

SELECT
    p.full_name              AS artist_display,
    COUNT(DISTINCT a.album_id) AS album_count
FROM   gold.dim_album           a
JOIN   gold.bridge_album_artist ba USING (album_id)
JOIN   gold.dim_person          p  USING (person_id)
WHERE  p.family_member = TRUE
GROUP  BY p.full_name
ORDER  BY album_count DESC
LIMIT  10;

Bob Marley leads the dynasty with 16 album credits in the catalog — counting both lead and group records. The second generation (Ziggy, Stephen, Damian, Ky-Mani, Julian) collectively outpaces Bob's own output, which is the whole point of the lineage.
